# Wholesale Customers K-Means Clustering

## About Dataset
- Source: https://archive.ics.uci.edu/dataset/292/wholesale+customers
- 440 customers, 8 features
- Spending data across 6 food categories

## What This Project Does
- Explores wholesale customer spending patterns
- Removes outliers from data
- Uses Elbow Method to find optimal K
- Groups customers into 5 clusters using K-Means
- Predicts cluster for new customers using KMeans and KNN

## Libraries Used
- pandas
- numpy
- matplotlib
- seaborn
- sklearn

## Results
- Optimal K = 5 clusters found using Elbow Method
- 5 distinct customer groups identified by spending behavior
